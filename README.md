# Global Weather Forecast application

## Project Description
GlobalCast is my first fully interactive web application, built to develop hands-on experience with modern JavaScript (ES6+), React, and real-world API integration. The app allows users to search for any city and instantly receive live weather data including today's conditions and the next four days.
This project was built as part of my personal portfolio to demonstrate skills in frontend development, asynchronous JavaScript, and UI design. The goal of this project is to deepen understanding of interactive web applications, API integration, and potentially React-based UI development.
